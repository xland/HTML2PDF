# HTML2PDF

Convert HTML to PDF without browser

# Features

- Does not rely on a browser
- Very fast performance
- Extremely low CPU and memory usage
- Supports most HTML5 and CSS3 features
- Single executable file (8M)
- [Test HTML](./index.html) , [Result PDF](./out.pdf)
